# Blitz the Gap: a nation-wide effort to guide citizen science toward the needs of biodiversity science

Repository to host scripts for the manuscript "Blitz the Gap: a nation-wide effort to guide citizen science toward the needs of biodiversity science".

**Data availability:** 
The iNaturalist Canada dataset was downloaded on December 1, 2025 and provided by the Canadian Wildlife Federation. These data are available from the iNaturalist API or upon request for large batch (>300k observations) downloads. 

**Code**
Scripts to reproduce analyses and figures in the manuscript are in the `scripts_ms/` folder.
